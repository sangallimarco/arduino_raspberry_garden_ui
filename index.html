Arduino raspberry garden ui

Flask UI to drive arduino ethernet or raspberry

Download this project as a .zip file Download this project as a tar.gz file

ARDUINO/RASPBERRY PI GARDEN

Requirements

  • pip install Flask
  • pip install Requests

Configuration

copy the main.cfg.template to main.cfg and choose the right engine

Engines:

  • arduinosimple : simple solenoid valve (on open , off closed)
  • arduino : first pin in the list is used to drive the bridge (solenoid that requires current inversion to change state)
  • raspberry : please refer to GPIO pinout and buy a 3.3V signal relay
  • dummy : test purpose only

Forecast Engine:

  • uk
  • it

Arduino

If you use Arduino to drive your valves please use the .ino provided and amend the ip setting

Let's Start

python main.py

now open a browser and go to address http://localhost:8080
